## 📋 Overview

Pawsome is a beautifully designed, fully responsive e-commerce website dedicated to cat lovers. The website showcases a variety of cat-themed merchandise including t-shirts, hoodies, mugs, accessories, and more. Built with pure HTML5, CSS3, and JavaScript, it features a modern design with smooth animations, interactive shopping cart functionality, and a complete user experience.

## ✨ Features

### 🎨 Design & UI
- **Modern Gradient Design** - Beautiful pink gradients throughout the site
- **Fully Responsive** - Works perfectly on desktop, tablet, and mobile devices
- **Smooth Animations** - Fade-in effects, hover animations, and floating elements
- **Sticky Header** - Navigation bar stays visible while scrolling
- **Back to Top Button** - Easy navigation on long pages

### 🛍️ E-commerce Features
- **Product Catalog** - Grid layout displaying products with images, prices, and ratings
- **Shopping Cart** - Add/remove items, update quantities, calculate totals
- **Product Details Page** - Image gallery with thumbnail selector
- **Local Storage** - Cart persistence across page refreshes
- **Toast Notifications** - User feedback for cart actions

### 📄 Pages Included
- **Home Page (index.html)** - Hero section, featured products, banners
- **Shop Page (shop.html)** - Complete product catalog with pagination
- **Product Page (sproduct.html)** - Detailed product view with image gallery
- **Blog Page (blog.html)** - Articles about cat care and products
- **About Page (about.html)** - Company information and app download
- **Contact Page (contact.html)** - Contact form and Google Maps integration
- **Cart Page (cart.html)** - Shopping cart management

### ⚡ Interactive Features
- Add to Cart functionality
- Quantity adjustment in cart
- Newsletter subscription
- Contact form with validation
- Mobile hamburger menu
- Scroll reveal animations
- Product image gallery with click-to-change
